# Food me!
## Instructions
To launch the application:<br/>
1 - Download the entire project<br/>
2 - Make sure Node.js and Angular v19+ are installed<br/>
3 - Open any terminal, navigate to the frontend folder, and run "ng serve -o"<br/>
